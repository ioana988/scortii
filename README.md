# 🎮 GameHub - Play Any Game You Want

A feature-rich gaming website with 6 classic and modern games, all playable in your browser!

## 🎯 Available Games

### 1. **Snake** 🐍
- Classic snake game with arcade gameplay
- Eat food to grow your snake
- Avoid walls and yourself
- **Controls:** Arrow keys to move

### 2. **Tic Tac Toe** ⭕
- Play against an AI opponent
- Win by getting 3 in a row
- Simple yet challenging
- **Controls:** Click cells to place your mark

### 3. **2048** 2️⃣
- Merge tiles to reach 2048
- Combine identical numbers
- Track your score and best score
- **Controls:** Arrow keys to move tiles

### 4. **Memory Game** 🧠
- Match pairs of emoji cards
- Test your memory skills
- Complete all 8 matches to win
- **Controls:** Click cards to flip them

### 5. **Rock Paper Scissors** ✂️
- Play against the computer
- Best of many rounds
- Score tracking included
- **Controls:** Click buttons to make your choice

### 6. **Flappy Bird** 🐦
- Navigate through obstacles
- Classic endless-runner style
- Beat your high score
- **Controls:** Click or press Space to flap

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Click on any game card from the menu
3. Follow the on-screen instructions
4. Click "Back" to return to the game menu
5. Play multiple games in one session!

## 💾 Features

- **Responsive Design** - Works on desktop and mobile devices
- **Score Tracking** - Keep track of your best scores
- **Pause Functionality** - Pause and resume games
- **Beautiful UI** - Modern gradient design with smooth animations
- **No Dependencies** - Pure HTML, CSS, and JavaScript
- **Local Storage** - Best scores are saved in your browser

## 📱 Browser Compatibility

- Chrome/Edge (Recommended)
- Firefox
- Safari
- Any modern browser supporting Canvas API

## 🎮 Game Tips

- **Snake:** Go for steady movements, avoid rushing
- **Tic Tac Toe:** Take center for best strategy
- **2048:** Move consistently in one direction first
- **Memory:** Start with edge cards for reference
- **RPS:** Predict computer patterns if any
- **Flappy Bird:** Time your flaps carefully

## 🛠️ Technical Stack

- **HTML5** - Structure and Canvas elements
- **CSS3** - Styling with gradients and animations
- **Vanilla JavaScript** - All game logic
- **LocalStorage API** - Score persistence
